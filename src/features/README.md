# Features

This folder is **feature-based** (one folder per use case). We are currently implementing UC-01..UC-06 with **no APIs** and **mock persistence** via `src/mockDb`.

Planned mapping:

- `auth` (UC-01): register/log-in/logout (mock)
- `quotes` (UC-02): quote form + pricing (mock)
- `bookings` (UC-03): confirm booking + deposit (mock)
- `feedback` (UC-04): feedback forms (mock)
- `scheduling` (UC-05): manager availability + time slots (mock)
- `inventory` (UC-06): inventory CRUD (mock)
